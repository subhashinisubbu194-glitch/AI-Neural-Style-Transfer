

# AI Neural Style Transfer

## Overview

The AI Neural Style Transfer project is a Python-based application that applies artistic style effects to photographs using image processing techniques. The application allows users to upload images, apply AI-style artistic transformations, and save the generated styled images through a user-friendly graphical interface.

This project was developed using OpenCV and Tkinter as part of an Artificial Intelligence Internship Task.

## Features

- Upload images from local system
- Apply artistic AI-style effects
- Real-time image transformation
- Save styled output images
- Interactive GUI interface
- Lightweight and beginner-friendly
- Fast image processing
